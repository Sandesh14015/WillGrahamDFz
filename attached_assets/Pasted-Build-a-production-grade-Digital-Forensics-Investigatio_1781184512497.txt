Build a production-grade Digital Forensics Investigation Platform called ForensiX.

The application must work both as:

A Linux/Windows command-line tool (CLI)
A Web-based investigation dashboard

Both interfaces must use the same shared backend modules.

Core Goal

ForensiX is a digital forensics and incident investigation platform that allows investigators to collect, analyze, correlate, and report digital evidence from files, logs, devices, emails, metadata, and disk images while maintaining forensic integrity.

The system should be modular, extensible, and suitable for cybersecurity investigations.

Forensic Principles

The system must always:

Work on copies of evidence
Never modify original evidence
Generate SHA256 hashes for integrity
Maintain chain-of-custody logs
Log every forensic action
Timestamp all operations
Be reproducible
